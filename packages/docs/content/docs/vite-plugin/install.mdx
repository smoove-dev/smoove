---
title: Install
description: Add @smoove/vite to your Vite config as a dev dependency.
---

`@smoove/vite` is a build-time plugin, so install it as a dev dependency.
`vite` (5 or newer) is a peer dependency you already have.

```bash
npm install -D @smoove/vite
# or
pnpm add -D @smoove/vite
```

Add it to your Vite config:

```ts
// vite.config.ts
import { smoove } from "@smoove/vite";
import { defineConfig } from "vite";

export default defineConfig({
  plugins: [smoove()],
});
```

That single call turns on both [jobs](/docs/vite-plugin/jobs). Each is
independent: if your project only does one of them, the other is a no-op.
