---
title: Install
description: Add @konva-motion/vite to your Vite config as a dev dependency.
---

`@konva-motion/vite` is a build-time plugin, so install it as a dev dependency.
`vite` (5 or newer) is a peer dependency you already have.

```bash
npm install -D @konva-motion/vite
# or
pnpm add -D @konva-motion/vite
```

Add it to your Vite config:

```ts
// vite.config.ts
import { konvaMotion } from "@konva-motion/vite";
import { defineConfig } from "vite";

export default defineConfig({
  plugins: [konvaMotion()],
});
```

That single call turns on both [jobs](/docs/vite-plugin/jobs). Each is
independent: if your project only does one of them, the other is a no-op.
